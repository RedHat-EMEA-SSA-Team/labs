## Install ssh public keys from all tiger's

```
curl -L https://raw.githubusercontent.com/RedHat-EMEA-SSA-Team/labs/master/authorized_keys >> ~/.ssh/authorized_keys
```

Or shortend
```
curl -L https://red.ht/tigerskeys >> ~/.ssh/authorized_keys
```
